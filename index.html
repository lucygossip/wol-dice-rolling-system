World of Legends - Dice Rolling System

General Information

Rules

  • Users must define their weapons, armour and animals before the fight. These must be defined within with the permitted weapon combinations. In order to define 'Magic', a user must be at the rank 'Wanderer' or higher.
  • Special weapons and animals MUST be in the inventory and BE SEEN, otherwise we will think they imagined it.
  • Users can attack once per weapon slot. A shield does not count as a weapon, however a shield can add defence against an opponent's attack.
  • Users start with a set amount of health. If it reaches zero, they are out of the fight but NOT dead. (unconscious/fled/rescued)
  • If a user ranks up during a fight, their OLD rank will be used till the end of the fight.
